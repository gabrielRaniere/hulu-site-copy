# Hulu UI Clone

Este projeto é uma **cópia da interface do site da Hulu**, desenvolvida com o objetivo de **praticar HTML e CSS**.
A aplicação não possui funcionalidades reais de streaming e foi criada apenas para fins educacionais e de treinamento em desenvolvimento front-end.

##  Objetivo

O principal objetivo deste projeto foi praticar:

* Estruturação de páginas com **HTML**
* Estilização com **CSS**
* Organização de layout
* Responsividade básica
* Reprodução visual de interfaces reais

##  Tecnologias Utilizadas

* HTML5
* CSS3

## 🎨 Funcionalidades

* Layout inspirado na página inicial da Hulu
* Seções organizadas com banners e cards
* Estilização semelhante à interface original
* Estrutura de página moderna usando HTML semântico

⚠️ **Observação:**
Este projeto é apenas um **clone visual**. Nenhuma funcionalidade real do serviço da Hulu foi implementada.

## 📂 Estrutura do Projeto

```
hulu-clone/
│
```

##  Como Executar

1. Baixe ou clone o repositório:

```bash
git clone https://github.com/seu-usuario/hulu-clone.git
```

2. Abra o arquivo `index.html` no navegador.
